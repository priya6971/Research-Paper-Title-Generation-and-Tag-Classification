# Research-Paper-Title-Generation-and-Tag-Classification
A transformer based deep learning approach to title generation of research papers and paper tags classification.

# Dependencies
1. PyTorch Cuda 11.1
2. Transformers 11.0
3. NLTK
4. Sumy
5. Seaborn
